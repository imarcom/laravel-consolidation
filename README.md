# Consolidation

## Installation
```json
{
    "require-dev": {
        "imarcom/laravel-consolidation": "dev-master"
    },
    "repositories": [
        {
            "type": "vcs",
            "url":  "git@bitbucket.org:imarcom/laravel-consolidation.git"
        }
    ]
}
```

```
php artisan vendor:publish --provider="Imarcom\Consolidation\ConsolidationServiceProvider" --tag="config"
```
